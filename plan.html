CIS 89A: Web Site Idea by Noah Cardoza

BOT.NET

Battle of the Internet is a game I and a friend came up with. The goal of the game is to hack other players, gain money, and reputation in the hacker underworld.
When all is done we plan to feature an in-game Terminal where most of the game will be played. While quite unrealistic, we aim to bring the most practical experience yet. We hope that this game will educate its players teaching them the basics of the UNIX OS and dangers of malicious users.

BOT.NET Web Site Plan


Home This will welcome the user and introduce them to the site and game play etc.
Contact This will hold a form element where people can send us requests and ideas.
Updates This will contain news about development and what not.
Register This is where people can register their email to receive automatic email updates.
Login This button will be faded out until an alpha version is released.
Ranking (This won't be relevant until an alpha version is released.) This page will contain a table of users sorted by their reputation. When scrolling to the bottom of the page, new users will be loaded. I might add a search bar, but that won't be fully operational till we host this on a server which supports PHP.
